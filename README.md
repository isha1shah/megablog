# MegaBlog 📝

MegaBlog is a modern and responsive blogging platform built with **React** and powered by **Appwrite** as the backend.  
It provides an easy way for users to create, edit, and manage blog posts with a rich text editing experience using **TinyMCE**.  

## 🚀 Features
- ✍️ Rich text editor powered by TinyMCE  
- 📰 Create, update, and delete blog posts  
- 🔐 User authentication with Appwrite  
- 📱 Fully responsive UI (works on mobile & desktop)  
- ⚡ Fast frontend built with React  

## 🛠 Tech Stack
- **Frontend:** React, TinyMCE, TailwindCSS   
- **Backend:** Appwrite (Authentication, Database, Storage)  
- **Deployment:** GitHub Pages (Frontend), Appwrite Cloud / Self-hosted
